- Instalar node, instalar npm
- criar diretório
- criar arquivo server.js
- criar um package.json: npm init -y  
- Alterar o tipo no package.json:  adicionar a linha  "type": "module",
- Comando para recarregar o server automáticamente ao savar: node --watch --no-warnings server.js  
- Comando para rodar script de dev: npm run dev
- se der erro de permissão para executar scripts: 
	- Abra powershell como admin, 
	- Rode: Get-ExecutionPolicy
	- Se Restricted:
		- Rode: Set-ExecutionPolicy RemoteSigned
		- Confirmer

- Instalar fastify, micro framework com funcionalidades essenciais, como roteamento: 
	- npm install fastify
- Instalar extenção REST Client no VSCode, para testar chamadas de api
- Instalar extenção DotENV para configurar arquivo de environment
- Criar arquivo .env

- Se aparecer erro `28P01` (falha de autenticação no PostgreSQL):
	1. Verifique se o .env está correto (pode copiar de .env.example)
	2. Se usar DATABASE_URL, encode caracteres especiais da senha (ex.: `*` vira `%2A`)
	3. Se o banco foi iniciado antes com outra senha, remova o volume e suba de novo:
		- docker compose down -v
		- docker compose up -d
